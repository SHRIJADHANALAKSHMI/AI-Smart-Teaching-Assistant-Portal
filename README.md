# Smart Teaching Assistant Portal

An AI-powered platform that helps faculty members create lecture materials, summaries, assessments, and learning roadmaps from uploaded syllabus documents, textbooks, notes, and presentations.

## Features

- College, Department, and Subject Management
- Professor Invitation System
- Syllabus and Material Upload
- AI-Generated Summaries and Notes
- Important Topic Identification
- Lecture Planning with Time Estimation
- Quiz and Assessment Generation
- Career Mapping and Learning Roadmaps
- Subject-Specific AI Chatbot

## Tech Stack

### Frontend

- React
- Vite
- Tailwind CSS

### Backend

- Java 21
- Spring Boot 4.1.0
- Spring Security
- JWT Authentication

### Database

- PostgreSQL

### AI

- Ollama
- DeepSeek / Llama 3
- LangChain4j.

## Getting Started

### Prerequisites

Install the following:

- Node.js (v20+)
- Git
- VS Code

### Clone Repository

```bash
git clone <repository-url>
```

```bash
cd smart-teaching-assistant-frontend
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Open in Browser

```text
http://localhost:5173
```

- ES7+ React/Redux Snippets
